# Secure Authentication Example

This repository contains a simple Python user authentication script used to teach secure software architecture and GitHub workflows.

## How to Use

1. Clone or fork this repository.
2. Create a new branch for your changes.
3. Make modifications, commit your changes, and push your branch.
4. Open a pull request to propose your changes.
5. Avoid committing passwords or sensitive data directly into the repository.
6. Use environment variables or config files to manage secrets securely.

## Important Notes
- This example contains hardcoded passwords on purpose for teaching. In real projects, **never** hardcode passwords.
- Peer review pull requests for security improvements!
